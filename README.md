Source code of the GitHub User's Info application:
[https://github-user-info.firebaseapp.com/](https://github-user-info.firebaseapp.com/)

## Setup
Clone this:
```
$ git clone git@github.com:matheusml/github-user-info.git
```

Install node deps:

```
$ npm install
```

And bower deps:
```
$ bower install
```

Run the tests:
```
$ npm test
```

Run the app:
```
$ npm start
```
