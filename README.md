# GitHub User's Info

Source code of the GitHub User's Info application

## Setup
Clone this:
```
$ git clone git@github.com:matheusml/github-user-info.git
```

Install node deps:

```
$ npm install
```

And bower deps:
```
$ bower install
```

Run the tests:
```
$ npm test
```

Run the app:
```
$ npm start
```
